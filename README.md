# User-Page

Personal portfolio page for Yanxin Huang, focused on medical image analysis,
computer vision, and machine learning research.

Live site: https://huangyanxin-china.github.io/User-Page/

## Overview

This repository contains a static portfolio website built with HTML, CSS, and
JavaScript. The page introduces research interests, selected projects,
publications, skills, and contact information.

## Featured Topics

- Brain tumor segmentation and medical image generation
- Multi-contrast cardiac MRI reconstruction
- Diffusion-guided segmentation with missing imaging modalities
- Medical image processing and computer vision

## Project Structure

| Path | Purpose |
| --- | --- |
| `index.html` | Main portfolio page |
| `src/style.css` | Page styling |
| `src/script.js` | Small interactive behavior |
| `assets/images/` | Profile, model, and result images used by the page |

## Deployment

The site is designed to be served with GitHub Pages from the `main` branch.

## Maintenance Notes

- Keep project images compressed before committing.
- Avoid committing editor-specific folders such as `.idea`.
- Update publication links and project descriptions as research work evolves.
