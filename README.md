# Yanxin Huang Portfolio

![Medical AI research banner](assets/images/research-banner.png)

Static portfolio site presenting Yanxin Huang's research direction in medical
AI, computer vision, medical image segmentation, and MRI reconstruction.

Live site: https://huangyanxin-china.github.io/User-Page/

## Positioning

This site is designed as a compact research portfolio rather than a generic
personal homepage. It highlights:

- Medical image segmentation under incomplete data conditions
- Brainstem glioma MR image synthesis and segmentation
- Multi-contrast cardiac MRI reconstruction
- Diffusion-guided and self-supervised learning themes

## Structure

| Path | Purpose |
| --- | --- |
| `index.html` | Main portfolio page |
| `src/style.css` | Visual system, responsive layout, and page styling |
| `src/script.js` | Lightweight page behavior |
| `assets/images/` | Research banner, portrait, project figures, and result images |

## Deployment

The site is served with GitHub Pages from the `main` branch.
